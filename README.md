# 🍔 Swiggy Clone (React)

A responsive food delivery web application inspired by Swiggy. The project demonstrates frontend development skills by consuming public Swiggy APIs, managing application state with Redux Toolkit, and building reusable React components.

> **Note:** This project is developed for learning and portfolio purposes only. It is not affiliated with or endorsed by Swiggy.

---

## ✨ Features

- Restaurant Listing
- Restaurant Details
- Dynamic Menu Page
- Search Restaurants
- Top Rated Restaurant Filter
- Add to Cart
- Remove Cart Items
- Cart Count using Redux Toolkit
- Help & Support Page
- Login & Signup UI
- Responsive Design
- API Integration
- Dynamic Routing

---

## 🛠 Tech Stack

- React 19
- Vite
- React Router DOM
- Redux Toolkit
- React Redux
- Tailwind CSS
- JavaScript (ES6+)
